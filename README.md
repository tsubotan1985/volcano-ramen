# 旭川らあめん「火山らぁめん」再現レシピ（推測）

茨城県（水戸・つくば）の名店「ひでまる」「やまる」の看板メニュー、火山らぁめんの再現レシピです。
中華鍋を使用した「あおり」調理法で、独特の香ばしさと乳化スープを再現します。

## 特徴
- **スタイル**: 旭川ラーメン（白味噌ベース）
- **特徴**: 濃厚な動物系スープ × 甘辛肉味噌 × レモンの酸味
- **調理法**: 中華鍋で味噌を焼き、スープを乳化させる「焼き味噌・あおり」製法

---

## 材料（1人前）

### A. 具材 & トッピング
- **もやし**: 1/2袋（必須）
- **玉ねぎ**: 1/4個（スライス）
- **長ネギ**: 白髪ネギにする（たっぷりと）
- **コーン**: 大さじ2
- **チャーシュー**: 1〜2枚
- **レモンスライス**: 1〜2枚（**※最重要の隠し味**）
- **中華麺**: 中細〜中太のちぢれ麺（低加水麺推奨）

### B. 火山肉味噌（辛味の核）
- **豚ひき肉**: 50g〜80g
- **豆板醤**: 大さじ1（辛さ調節）
- **甜麺醤**: 大さじ1/2
- **砂糖**: 小さじ1
- **おろしニンニク**: 少々
- **ごま油**: 適量

### C. スープ調味
- **ラード**: 大さじ1.5（チューブ可。サラダ油不可）
- **市販の味噌ラーメンスープ**: 1袋（白味噌系）
- **豆板醤**: 小さじ1（炒め用）
- **おろしニンニク**: 1片分（潰す）
- **豚骨/鶏ガラスープ**: 300-350ml（お湯の代わりにスープで割ると濃厚さUP）

---

## 調理手順

### 1. 「火山肉味噌」を作る（事前準備）
フライパンでひき肉をごま油で炒める。色が変わったら調味料B（豆板醤、甜麺醤、砂糖など）を入れ、水分が飛んでポロポロになるまでしっかり炒める。
* **ポイント**: 甘辛く仕上げることで、スープに溶けた時のコクになる。

### 2. 中華鍋のプレヒート
中華鍋（または鉄フライパン）に**ラード**と潰したニンニクを入れ、強火にかける。
* **ポイント**: うっすら煙が出るまでカンカンに熱する（大火力の再現）。

### 3. 野菜の「焼き」炒め
もやしと玉ねぎを一気に投入し、あまり動かさずに一瞬「焦げ目」をつけるイメージで炒める。

### 4. 「焦がし味噌」テクニック（重要）
野菜を端に寄せ、空いた鍋肌（一番熱い部分）に**味噌ダレ**と**豆板醤（炒め用）**を直接投入する。
* **ポイント**: スープで溶く前に、ラードで味噌を10秒ほど「揚げる（焼く）」。これで生味噌の臭みが消え、お店のロースト香が出る。

### 5. スープの乳化
味噌が香ばしく焼けたら、熱いスープ（または熱湯）を一気に注ぐ。「ジュワッ！」と蒸気が上がったら、そのまま**強火で1〜2分ボコボコ沸騰**させる。
* **ポイント**: 沸騰させることでラードとスープが乳化し、トロっとした口当たりになる。

### 6. 盛り付け
1. 茹でた麺を丼に入れる。
2. 中華鍋の中身（スープと野菜）をすべて注ぐ。
3. 中央に野菜を盛り、その頂上に「1.の肉味噌」を山盛りにする。
4. 白髪ネギ、コーン、チャーシューを添える。
5. 最後に**レモンスライス**を乗せる。
